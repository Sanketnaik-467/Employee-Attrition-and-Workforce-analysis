# Power-BI-KPI-Dashboard
# 📊 Finance KPI Dashboard | Power BI

## Overview

This project showcases an interactive **Bank Loan Portfolio Performance Dashboard** built using **Microsoft Power BI** to analyze and monitor key lending and portfolio performance metrics. The dashboard provides actionable insights into loan applications, funded amounts, repayments, portfolio quality, interest rates, and borrower characteristics, helping stakeholders make data-driven lending decisions.

## Business Challenge

One of the major challenges during this project was understanding banking and loan portfolio metrics while creating meaningful KPIs and DAX measures. The objective was to transform raw loan data into a user-friendly dashboard that could effectively track portfolio performance, identify risk patterns, and support business decision-making.

## Solution

By leveraging Power BI, DAX calculations, and data visualization techniques, I developed an interactive dashboard that provides a comprehensive view of loan portfolio performance. The dashboard enables users to analyze performing and non-performing loans, evaluate lending trends, and explore borrower and loan characteristics through dynamic filters and visualizations.

## Key Features

• Total Funded Amount & Amount Received Tracking
• Average Interest Rate Monitoring
• Performing vs Non-Performing Loan Portfolio Analysis
• Loan Status Performance Evaluation
• Monthly Loan Application Trend Analysis
• State-wise Loan Distribution using Geographic Mapping
• Loan Purpose Analysis
• Home Ownership Analysis
• Loan Term Analysis
